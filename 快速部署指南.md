### 第 1 步：获取 GitHub Token（3 分钟）

**在你本地浏览器操作**：

```
1. 访问：https://github.com/settings/tokens

2. 登录 GitHub 账号：haohaoxuebi1

3. 点击 "Generate new token (classic)"

4. 填写信息：
   - Note: yuezhiyun-deployment
   - Expiration: 90 days（或 No expiration）
   - 勾选权限：✅ repo（全选）

5. 点击底部的 "Generate token"

6. 复制 Token（以 ghp_ 开头）
   ⚠️ 只显示一次，一定要复制！
```

### 第 2 步：运行推送脚本（1 分钟）

**在终端执行**：


**然后**：

### 第 3 步：在 Vercel 部署（3 分钟）

**在你本地浏览器操作**：

```
1. 访问：https://vercel.com

2. 点击 "Login" → "Continue with GitHub"
   - 授权 Vercel 访问你的 GitHub

3. 登录后，点击 "Add New Project"

4. 选择 "Import Git Repository"

5. 在列表中找到 "yuezhiyun-beauty"
   - 如果看不到，点击 "Adjust GitHub App Permissions"
   - 授权访问 haohaoxuebi1 的仓库

6. 找到后点击 "Import"

7. 配置保持默认（已自动识别）：
   - Framework Preset: MkDocs
   - Build Command: mkdocs build
   - Output Directory: site

8. 点击 "Deploy"

9. 等待 1-2 分钟，看到绿色对勾 ✅

10. 获得访问地址：
    https://yuezhiyun-beauty.vercel.app
```

## 🌐 可选：绑定自定义域名

如果你有域名（如 yuezhiyunbeauty.com），可以绑定：

### 在 Vercel 添加域名

```
1. 进入项目 → Settings → Domains

2. 点击 "Add"

3. 输入：
   - yuezhiyunbeauty.com
   - www.yuezhiyunbeauty.com
```

### 在阿里云配置 DNS

```
1. 访问：https://www.aliyun.com
2. 登录 → 控制台 → 域名控制台
3. 找到域名，点击 "管理"
4. 点击 "解析设置"
5. 添加两条记录：

记录 1 - CNAME:
类型：CNAME
主机记录：www
记录值：cname.vercel-dns.com
TTL：10 分钟

记录 2 - A 记录:
类型：A
主机记录：@
记录值：76.76.21.21
TTL：10 分钟
```

**等待 5-30 分钟 DNS 生效即可！**

## 🆘 遇到问题？

### 问题 1：找不到 Token 设置页面
```
解决：直接访问 https://github.com/settings/tokens
```

### 问题 2：脚本提示 Permission denied
```
解决：在终端执行
chmod +x ~/openclaw/workspace/yuezhiyun-beauty/push-to-github.sh
然后重新运行脚本
```

### 问题 3：Vercel 找不到仓库
```
解决：
1. 在 Vercel 登录页面点击 "Adjust GitHub App Permissions"
2. 授权 Vercel 访问 haohaoxuebi1 的仓库
3. 刷新页面重新导入
```

### 问题 4：推送失败
```
可能原因：
1. Token 无效 → 重新生成 Token
2. Token 权限不足 → 确保勾选了 repo 权限
3. 网络问题 → 检查网络连接
```
