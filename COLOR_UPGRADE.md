## 🌟 设计细节优化

### 1. 阴影效果
```css
box-shadow: 0 6px 20px rgba(201, 169, 98, 0.12);
```
- 金色阴影，更柔和
- 多层阴影，更立体

### 2. 渐变效果
```css
background: linear-gradient(135deg, #c9a962 0%, #b0924a 100%);
```
- 香槟金渐变
- 45 度角，更自然

### 3. 动画效果
```css
animation: gradientShift 8s ease infinite;
```
- 慢速渐变，更优雅
- 8 秒周期，不急躁

### 4. 圆角设计
```css
border-radius: 20px;
```
- 大圆角，更现代
- 统一风格，更协调

## 🎨 配色代码

### CSS 变量
```css
:root {
  --md-primary-fg-color: #c9a962;      /* 香槟金 */
  --md-accent-fg-color: #6b4c7a;       /* 深紫色 */
  --color-rose-gold: #e0b4a6;          /* 玫瑰金 */
  --color-soft-gold: #fdfbf7;          /* 米白色 */
}
```

### 使用建议
- 主色使用 60%
- 辅助色使用 30%
- 强调色使用 10%
